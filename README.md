# The Urban Green

A modern, responsive eCommerce UI built with Next.js and Tailwind CSS for a premium plant brand.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Pages

- `/` Home page
- `/shop` Shop page
- `/shop/monstera-deliciosa` Example product page

## Project structure

```text
app/
  layout.tsx
  page.tsx
  shop/
    page.tsx
    [slug]/
      page.tsx
components/
  care-tabs.tsx
  instagram-grid.tsx
  product-card.tsx
  product-grid.tsx
  section-heading.tsx
  size-selector.tsx
data/
  instagram.json
  products.json
lib/
  data.ts
  types.ts
public/
  instagram/
  products/
```
